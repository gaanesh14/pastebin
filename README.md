PasteLink is a full-stack Pastebin-style web application that allows users to create, store, and share text snippets using unique links.  
All generated paste links are automatically stored in a personal **My Paste Links** dashboard for easy reuse.

 Features
 
 Create text pastes instantly
 Generate **shareable links**
 Automatically store all generated links
 Optional expiration time (minutes)
 Optional max view limit
 Personal **My Paste Links** dashboard
 Clean, professional UI using Tailwind CSS

---
 Tech Stack

### Frontend
- React
- React Router
- Axios
- Tailwind CSS
### Backend
- Node.js
- Express.js
- MongoDB (or in-memory storage if applicable)
- NanoID (for short IDs)

### Deployment
- Frontend: Render
- Backend: Render

frontend setUP: 
cd frontend
npm install
npm run dev

backend setup:
cd backend 
npm install
npm start
